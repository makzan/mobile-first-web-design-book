## Customizing jQuery mobile


First, please check out the [demo page of the default theme](http://demos.jquerymobile.com/1.4.3/theme-default/).

jQuery define theme by using English letter. In theory we have 26 sets of color theme to set. To define our own color scheme, we can go to the [Theme Roller tool](http://themeroller.jquerymobile.com/).
